## start

gradle
```groovy
implementation 'net.cinling:springboot-lib:{version}'
```

maven
```xml
<dependency>
  <groupId>net.cinling</groupId>
  <artifactId>springboot-lib</artifactId>
  <version>{version}</version>
</dependency>
```
